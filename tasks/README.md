# RL DQN templates

Ovo su starter fajlovi za zadatke iz `src/3. RL.ipynb`.

`common.py` je pomoćni fajl koji možete koristiti bez menjanja. Ostali fajlovi
sadrže TODO mesta koja treba dopuniti:

- `q_learning.py` - Q-learning klasa za FrozenLake
- `problem_3_one_hot_dqn.py` - one-hot wrapper + DQN trening
- `problem_4_custom_mlp_dqn.py` - custom PyTorch feature extractor
- `problem_5_cartpole.py` - CartPole + DQN

Pokretanje:

```bash
python q_learning.py
python problem_3_one_hot_dqn.py
python problem_4_custom_mlp_dqn.py
python problem_5_cartpole.py
```
